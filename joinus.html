公司简介

 

联桥网云信息科技（长沙）有限公司（简称：联桥网云）是一家物联网技术服务公司，公司成立于2015年，本着以客户需求为创新目标，在这“万物互联网”时代，为传感器设备厂商，应用云平台开发商，解决方案提供商提供具有颠覆传统的运营模式，以及为终端用户提供有效的个性化服务。欢迎各界同仁前来参与，共展宏图。

 

招聘人数：

1、网络及应用软件工程师 2-3人

2、系统与软硬件设计高级工程师 1-2人

3、嵌入式软件工程师 1-2人

4、客户服务人员 2-3人

 

工作地：湖南省长沙市

职位简述

参与开发联桥网云的 web 应用程序、包括用户网站、应用云平台、web 服务器、以及移动应用软件。对我们的客户和联桥网云，可用性和软件功能及运行质量至关重要，联桥网云依靠web 应用程序获取、分析、和推送物联网设备数据，我们的客户将依靠web 应用程序来访问他们的监视设备和他们个人的数据记录，设定个性化的解决方案。用最新的技术和工具加上你的创造力，获得最好的体验，解决实际的问题。

 

职位责任

•      设计、构建、编写包括移动应用程序的 web 应用程序

•      围绕实时 web应用程序、解决各类问题

•      分析和改进包括浏览器的性能和应用程序的稳定性

•      衔接硬件传感器、以及和其他软件系统的整合

•      维护和支持 web 应用程序

•     独立完成公司产品的功能模块编码

•     负责公司平台相关业务模块的分析、设计

•     相关开发文档的整理与编写
